Instrucciones sobre el HTML a generar

Lee detenidamente las instrucciones antes de empezar.

Para realizar bien esta prueba deberás generar 2 Archivos HTML:

  1. Una página de tu CV con nombre de archivo index.html
  2. Una página con gifs divertidos con nombre de archivo galeria.html

En todo momento los archivos que generes deberán ir dentro de la carpeta que ya existe con tu nombre y apellidos, el html en la ruta iniciada bajo la carpeta html, el css bajo la ruta resources/css y las imágenes deberán estar descargadas en tu carpeta bajo la ruta resources/img

Recuerda que, en todo momento debes estar trabajando en la rama que has creado para tí y en tu fork creado del repositorio Github

Recuerda que:

  • Todo lo relativo a formato debe ir en tu archivo .css propio
  • Para los links, imágenes y css debes usar siempre rutas relativas
  • Las imágenes deben estar descargadas en tu carpeta img
  • Debes estar haciendo commits cada pocos cambios, haciendo push de vez en cuando y trabajando en la rama que has creado antes de empezar a codificar
  • Dispones de toda la ayuda de internet y los manuales del aula virtual
  • Una prueba de que algunas cosas han ido bien será que cuando acabes, el menú de esta página lleve hasta tus páginas HTML
